

			 ### Key Binds ###



		  	SUPER = Windows-Key

Bind		  	   // APPS \\				Effect

SUPER, Q			➜			Terminal
SUPER, E			➜			File Explorer
SUPER, R			➜			App Launcher
SUPER, Z			➜			Zen-Browser (Web-Browser)
SUPER, M			➜			Spotify
SUPER, S 			➜			Steam
SUPER, D			➜			Discord (Vesktop)
SUPER, N			➜			Opens a floating Note to edit

SUPER, SHIFT, T 		➜			Emergency Terminal (Floating Terminal)


Bind			// Windows Management \\		Effect

SUPER, K 			➜			Kills current acvtive Window
SUPER, SHIFT, K 		➜			Click window to force kill
SUPER, V 			➜			Toggles floating on the active window
SUPER, J			➜			Splits windows horizontally or vertically
SUPER, F 			➜			toggles fullscreen on active window
SUPER, ESC			➜			Shows logout screen (ESC to close)
SUPER, L 			➜			Lock the screen


SUPER, Arrow Keys		➜			Switch the current focused (active) window
SUPER, SHIFT, Arrow Keys	➜			Swap current active with the choosen one

PRINT(Druck)			➜			Screenshot whole current screen (Where an active window is)
SUPER, PRINT			➜			Screenshot the active window
SUPER, SHIFT, PRINT		➜			Select region to screenshot
